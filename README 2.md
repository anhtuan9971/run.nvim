## Installation

### Packer
```lua
use 'Manas140/run.nvim'
```

### Manual Installation
```sh
  git clone https://github.com/Manas140/run.nvim && cd run.nvim
  nvim --cmd "set rtp+=."
```

## Execute The Plugin
```
  Keybind - Control + E
  Command - :Run
```
